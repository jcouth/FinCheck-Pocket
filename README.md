## Instalação

```bash
git clone https://github.com/jcouth/FinCheck-Pocket.git
cd FinCheck-Pocket
yarn install

# iOS
cd ios && pod install
```

## Comandos Úteis

```bash
# Iniciar app - Android
yarn android

# Iniciar app - iOS
yarn ios

# Iniciar Storybook
yarn storybook

# Rodar testes unitários
yarn test

# Reset cache
yarn start --reset-cache
```
